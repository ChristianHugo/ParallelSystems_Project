Detailed Description

Parser um Konfigurations-Parameter auszulesen.

Liest mithilfe von regulären Ausdrücken Parameter aus einem String. Die Parameter müssen die Form

ParameterName=ParameterWert

haben. Bspw. execution_strategy=0 für eine sequentielle Ausführung. Ein gefundener Wert wird als boost::optional< T > zurückgegeben oder boost::None.

Leerzeichen sind nicht unterstützt, bei doppelten Parameter wird der erste genommen.
