Detailed Description

Basisklasse für alle Strategien.

Definiert die Basisklasse für Strategien aus dem Stratie Entwurfsmuster. Ableitende Klassen approximieren die Wellengleichung mit unterschiedlichern Ausführungsstrategien. Sie überschreiben den () operator als Einstiegspunkt.
