Detailed Description

barrier Synchronisierung für std::threads

Die Funktion wait() kann zum Synchronisieren z.B. am Ende einer Zeile aufgerufen werden. In der C++ Standard Library ist kein eigenes barrier vorhanden, jedoch basiert die Implementierung von StdThreadBarrier auf Mitteln aus den Headern: <condition_variable> und <mutex> der std.
