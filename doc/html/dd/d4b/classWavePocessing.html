Detailed Description

Klasse zum approximieren der Wellengleichung.

Die Klasse implementiert zentral die numerische Approximierung der Wellengleichung. Sie wird von den Ausführungsstrategien benutzt. Nachdem sie mit einem WaveProcessingParams Objekt instanziiert wurde, kann eine der process() Methoden aufgerufen werden.

Approximiert anhand der Koordinaten.
